## Install dependencies for React App-
```
cd client
npm install
```

## 1. Start the local ethereum blockchain-
```
truffle develop
```
  ## Compile the smart contracts-
  ```
     compile
 ```
  ## Deploy the smart contracts-
  ```
     migrate
  ```


## 2. Start the client side in another terminal-
```
cd client
npm run start
```
